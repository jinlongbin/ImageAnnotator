# ImageAnnotator
ImageAnnotator is a Python-based tool for easy and precise annotation of images

## Folder structure
The selected folder should be in the following structure
```
selected folder
|----single
|    |----b1
|    |    |----1.png
|    |    |----2.png
|    |    |...
|    |----b2
|    |    |----1.png
|    |    |----2.png
|    |    |...
|    |...
|    |----g1
|    |    |----1.png
|    |    |----2.png
|    |    |...
|    |----g2
|    |    |----1.png
|    |    |----2.png
|    |    |...
|    |...
|----group
|    |----1.png
|    |----2.png
|    |...
|----annotation.json
```
